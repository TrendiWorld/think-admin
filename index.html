THIRD PARTIES · WHAT LEAVES THINK
Server-side unless noted. API keys never ship in the app. Cache-first: repeat requests on stable content cost zero external calls.

WHAT WE COLLECT · stored in Supabase
Account
Handle, optional email (saved accounts), birth-year (age gate), sign-in identity
Content
Questions, comments, voice-story audio + transcripts, votes, reactions, optional political stance
Social
Follows, blocks, in-person meets, direct messages, like-minded scores
Usage
Sessions, screen views, daily AI/voice meters, Ripples ledger, country/locale (from device settings, no GPS)
Device
Push token (if opted in). No precise location, no contacts, no health/financial data
ACCOUNT DELETION · COMPLETE ERASURE ✓
Deleting an account (Settings → Delete account) runs delete_my_account then removes the auth user. It erases every piece of the above:
  • ~45 database tables deleted explicitly (not relying on cascades that a schema change could break)
  • All uploaded files (avatar, voice-story audio + covers) by the user's {uid}/ storage path — fixed 29 Jun 2026 after finding the old rule missed service-role uploads
  • The auth account + sessions/identities
Verified against the live schema: every table with a user foreign key is covered. Deletion request support: founder@think-app.com.
